# ABB Website UI Clone (Frontend Practice)

Bu layihə **ABB (Azərbaycan Beynəlxalq Bankı)** rəsmi veb-saytının ana səhifəsinin naviqasiya panelinin və genişləndirilmiş mobil menyusunun (responsive navigation) frontend klonudur.

## 🎯 Layihənin Məqsədi
Bu layihə heç bir kommersiya məqsədi daşımır. Əsas məqsəd:
- Real dünya layihələrindəki mürəkkəb naviqasiya strukturlarını təhlil etmək.
- **Tailwind CSS** ilə "Mobile-first" dizayn prinsiplərini tətbiq etmək.
- **Vanilla JavaScript** vasitəsilə dinamik interfeys elementlərini (məsələn, mobil menyu) idarə etmək.
- Portfoliom üçün peşəkar frontend bacarıqlarımı nümayiş etdirmək.

## 🛠 Texnologiyalar
- **HTML5**: Semantik və əlçatan struktur.
- **Tailwind CSS**: Sürətli və effektiv vizuallaşdırma.
- **JavaScript (ES6)**: Menyu funksionallığı və DOM manipulyasiyası.
- **FontAwesome / SVG**: Vektor ikonlar.

## 🚀 Özəlliklər
- **Tam Responsive**: Desktop, Tablet və Mobil cihazlar üçün tam uyğunlaşdırılmış interfeys.
- **Dinamik Mobil Menyu**: Sağ tərəfdən sürüşərək açılan və arxa fonun hərəkətini (scroll) donduran menyu sistemi.
